# Projet : Analyse des Performances Logistiques

## Description
Ce projet analyse les données d'une entreprise de transport pour :
- Visualiser les performances des transporteurs
- Prédire les risques de retard

## Structure du projet
- `data/` : Données brutes et nettoyées
- `notebooks/` : Analyse exploratoire et ETL
- `dashboard/` : Tableau de bord Power BI
- `models/` : Modèle de prédiction

## Technologies utilisées
- Python (Pandas, Scikit-learn)
- Power BI
- Git / GitHub
